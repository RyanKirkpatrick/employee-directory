Profiles

The directory allows you to view profiles of employees, rooms and printers. Profile filters allow you to focus on a particular individual or group to select from.

Employee Profiles

Employee profiles display information about individual employees and allow you to find out more information or communicate easily with an employee.

The employee profile card has the following options under the employee picture:

  • Opens a Lync chat session with the employee
  • Displays the employee in the Organization Chart
  • Displays the employee's desk location on a map of the office

Any groups which the employee is apart of (committees, developer guilds, etc.) will also have an icon displayed below their picture.

The employee profile card displays specific information about the employee including:

  • Name
  • Job Title
    • clicking the Job Title will display all employees with the same job title
  • Department
    • clicking the Department will display all employees in that department
  • Team
    • clicking the Team will display all employees on that team
  • Manager
    • clicking the Manager will display the manager's profile
  • Email
    • clicking the Email Address will begin to compose an email message to the employee
  • Internal Extension
  • Direct Dial Phone Number
  • Location
    • clicking the Location will display the employee's desk location on a map of the office

You can direct link to a particular individual or group using one of the following query strings:

  • First Name

    Passing in "firstname", you will get all employees with the specified first name.
    • http://directory.liazon.corp/employees/profile/?firstname=matt
    You can specify multiple first names by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/?firstname=matt,eric

  • Last Name

    Passing in "lastname", you will get all employees with the specified last name.
    • http://directory.liazon.corp/employees/profile/?lastname=laderer
    You can specify multiple last names by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/?lastname=land,laderer

  • First & Last Name

    Passing in "firstname" & "lastname", you will get all employees with the specified first & last name.
    • http://directory.liazon.corp/employees/profile/?firstname=chad&lastname=laderer
    You can specify multiple first & last names by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/?firstname=chad,connor,natalia&lastname=land,laderer

You can direct link to a particular individual or group using one of the following url patterns:

  • Job Title

    Specifying a "title", you will get all employees with the specified title.
    • http://directory.liazon.corp/employees/profile/title/sr. engineer, software
    You can specify multiple departments by separating them with pipe (|) in the query string
    • http://directory.liazon.corp/employees/profile/department/account executive|sr. engineer, software

  • Department

    Specifying a "department", you will get all employees in the specified department.
    • http://directory.liazon.corp/employees/profile/department/marketing
    You can specify multiple departments by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/department/finance,hr

  • Team

    Specifying a "team", you will get all employees on the specified team.
    • http://directory.liazon.corp/employees/profile/team/prestige worldwide
    You can specify multiple teams by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/team/prestige worldwide,00 Agents

  • Employee ID

    Specifying an "employee" with the employee's ID number, you will get a specific employee.
    • http://directory.liazon.corp/employees/profile/employee/1209546
    You can specify multiple employees by separating them with commas (,) in the query string
    • http://directory.liazon.corp/employees/profile/employee/1209546,1209654,1209764

Room Profiles

Room profiles display information about individual rooms and allow you to find out more information about a room.

The room profile card displays specific information about the room including:

  • Name (number)
  • Room Type
  • Internal Extension
  • Direct Dial Phone Number
  • URL for Display
  • Capacity
  • Location
    • clicking the location will display the room's location on a map of the office

You can direct link to a particular room or group of rooms using one of the following query strings:

  • Room Name

    Specifying a "name", you will get the room with the specified name.
    • http://directory.liazon.corp/rooms/profile/name/big ditch
    You can specify multiple rooms by separating them with commas (,) in the query string
    • http://directory.liazon.corp/rooms/profile/name/big ditch,main

  • Room Type

    Specifying a "type", you will get all rooms with the specified type.
    • http://directory.liazon.corp/rooms/profile/type/training
    You can specify multiple room types by separating them with commas (,) in the query string
    • http://directory.liazon.corp/rooms/profile/roomtype/training,break

Printer Profiles

Printer profiles display information about individual printers and allow you to find out more information about a printer.

The printer profile card displays specific information about the printer including:

  • Network Name
  • Color Type
  • Manufacturer and Model
  • Location
    • clicking the location will display the printer's location on a map of the office

You can direct link to a particular printer or group of printers using one of the following query strings:

  • Printer Name

    Specifying a "name", you will get the printer with the specified name.
    • http://directory.liazon.corp/printers/profile/name/LZB7P1-LEX510
    You can specify multiple printers by separating them with commas (,) in the query string
    • http://directory.liazon.corp/printers/profile/name/LZB7P1-LEX510,LZB8P2-BIZ280

  • Printer Brand

    Specifying a "brand", you will get all printers of the specified brand.
    • http://directory.liazon.corp/printers/profile/brand/lexmark
    You can specify multiple printer brands by separating them with commas (,) in the query string
    • http://directory.liazon.corp/printers/profile/brand/lexmark,Hewlett Packard
